# ZenFlow

ZenFlow is a minimalist workspace designed to help you enter a flow state by
eliminating digital clutter. It operates on the principle of "Single-Tasking."

## Features

- **Monotropic Focus:** A single input field for your current goal. No lists.
- **Ambient Soundscape:** Integrated audio toggles for White Noise, Rainfall, 
  and Cafe Ambience.
- **Deep Work Timer:** A visual Pomodoro timer that fades into the background.
- **Privacy First:** No accounts, no tracking. All data persists in LocalStorage.

## Tech Stack

- **Frontend:** HTML5, CSS3 (Modern Flexbox/Grid)
- **Logic:** Vanilla JavaScript (ES6+)
- **Icons:** Lucide-style SVG icons
- **Storage:** Browser LocalStorage

## Getting Started

1. Clone the repository.
2. Open `index.html` in any modern browser.
3. Type your main objective and hit "Start Flow."
